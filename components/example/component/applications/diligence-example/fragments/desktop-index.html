Diligence Examples

Welcome to Diligence!

This application is designed to help you learn how to use Diligence. It is designed as a collection of live examples for every single Diligence feature and service. Together with each example, we've also included the relevant chapter from the Diligence Manual, and provided a link to the example's source code, as well as the relevant API documentation.

To navigate, use the menu on the left. When reading the manual, a small floating menu will appear at the bottom right to help you move between sections.

There's a lot of information here, and it may be overwhelming! Where to start? This really depends on what has drawn you to Diligence in the first place. So, let make a few suggested itineraries:

Using Diligence to Create a CMS-Like Web Application

Diligence contains several building blocks for this. The most important is the Documents Service. You'll then want to look at the higher-level features, the Discussion Feature, the Wiki Feature and the Blog Feature. Also of immediate use are the Syndication Service and the Linkback Service.

Using Diligence to Create Social Networking Web Application

Start with the Authentication Service to see how you can allow social networking users to login to your application. Also check out Gravatar Integration.

Using Diligence to Create an Ext JS or Sencha Touch Web Application

This is a rich area of Diligence, and support for these client-side frameworks is very comprehensive. Start with the Sencha Integration page, and move form there to exploring the features you need. It will lead you to also exploring the REST Service and the RPC Service

Using Diligence to Create a RESTful API over MongoDB

You want to start with the REST Service, which can automatically create a URI-space around your MongoDB database.
