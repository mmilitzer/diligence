Log Panel

This panel at the bottom of the screen usually shows you the last lines of the log file and refreshes its view every 3 seconds.

You can toggle auto-refreshing using the "Freeze" button.

You can also travel backward and forward in the log using the arrow buttons. When you start traveling, auto-refresh automatically freezes, so you may want to un-toggle "Freeze" when you're done traveling.

Looking for something in particular in the log? Use the "Grep" area to enter a regular expression with which to filter the log file. (This works by grepping the log file into the server's temporary directory.) Grep stays active while auto-refreshing. It's live grep!

Switch to a different log file using the combo box at the bottom right, either choosing one of the available options or typing in your own. The path is relative to your container's "/logs/" directory.

Code Panel

Type in your JavaScript code with friendly syntax coloring, and click "Execute" to evaluate your code on the server.

By default all of the Diligence, Prudence, Sincerity and MongoDB namespaces are already available to you, so you can use APIs having to "document.execute" the libraries first.

The "logger" variable is initialized using Prudence's logging library, providing you with a quick way to see the results of your code right in the console's log panel: 

var name = 'World'
logger.info('Hello, {0}', name)

You can also return full textual representations to the console: simply set the "representation" variable to your value. If a representation is made available by your code, the console will popup a window with the result. Use newline characters ("\n") to return multiline representations: 

representation = 'Hello, World!\nHello, other world!'

As shortcuts, you can use the print() and println() functions to append anything to the representation. Arguments will be implicitly stringified.

Click "Execute and Download" to download this representation via your browser. By default, it is a plain text file named "console.txt", but you can change this using Prudence's standard API. For example, let's download a CSV file to be opened by spreadsheet software: 

println('1,2,3,4')
println('5,6,7,8')
println('9,10,11,12')
conversation.mediaTypeName = 'text/csv'
conversation.disposition.filename = 'transactions.csv'

Saved Programs Window

Click "Save Program" to store your program for later. Programs are stored in MongoDB's "programs" collection. Note that using the same program name again will override your stored program.

Double-click on a program name in the "Saved Programs" panel in order to load it into your code panel.

Click "Rename" to rename it. Click "Delete" to delete it.
